# 프로젝트 소개
 - 프로젝트 이름 : RuningGame
 - 개발언어 및 엔진 : C#, 유니티3D 
 - ## 게임 설명
    몰려오는 좀비를 죽이는 게임입니다. 아이템을 먹어 플레이어를 강화하거나, 동료를 늘려보세요!
